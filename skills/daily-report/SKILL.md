# Daily Report Skill

当触发时执行以下步骤：

## 1. 读取今日记忆文件

读取 `memory/YYYY-MM-DD.md`（今天和昨天的），提取：
- 今天完成的主要任务
- 重要决策
- 遇到的问题和解决方案

## 2. 格式输出

按以下格式组织内容：

```
## 📅 [日期] 日报

### ✅ 完成的工作
- 

### 💡 重要决策
- 

### 🕳️ 遇到的问题
- 

### 📋 明天计划
- 
```

## 3. 如果没有记忆文件

提示用户今天还没有工作日志，可以开始记录。

## 注意事项

- 语言风格：简洁专业，直接列出要点
- 不要废话，老板喜欢直接看结论
