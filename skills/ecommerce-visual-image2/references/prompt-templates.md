# Prompt Templates

## 10-Screen JD Detail Page Map

Use these as starting points. Replace bracketed text with product-specific details.

### 1. Hero Cover

```text
Use the provided product image as the strict visual reference. Generate a professional JD e-commerce detail page image, 9:16 vertical. Screen 1: premium hero cover for [product name]. [style direction], product centered large and sharp, natural commercial lighting, refined background elements matching the product tone. Preserve product shape, proportions, packaging color, logo area, label hierarchy, material finish, and key visible details as much as possible. Add short headline: [headline]. Subtitle: [subtitle]. High-end commercial e-commerce design, clean hierarchy, no clutter, no watermark.
```
